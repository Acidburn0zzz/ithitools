ITHI Project

In 2017, ICANN started a project to monitor the health of the registered identifiers ecosystem, through a set of Identifier Technology Health Indicators (ITHI), or ITHI Metrics. There are seven metrics:

  1. M1 tracks the accuracy of the identifiers' data
  2. M2 tracks the number of abusive domain name registrations
  3. M3 tracks the usage and abuse of the DNS root
  4. M4 tracks name usage and leakage by DNS clients
  5. M5 tracks the truthfulness of DNS responses
  6. M6 tracks the health of IANA registries for DNS parameters
  7. M7 tracks the deployment of DNS SEC.

This project is in development. We do not have available data yet for the metrics M1 and M5.

The measurements of M4 are based on DNS traces collected at National University of La Plata (UNLP), Argentina.
